# CartifyZone
Django E-Commerce Web Application

## Django E-Commerce Web Application

CartifyZone is a modern e-commerce web application developed using Django. The platform allows users to browse products, register accounts, log in securely, add products to cart, place orders, and manage their shopping experience.

### Features

* User Registration and Login
* Product Catalog
* Product Details Page
* Shopping Cart Management
* Checkout System
* Order Placement
* Order History Tracking
* Responsive User Interface
